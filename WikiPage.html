Determines whether this wiki page should appear on the public list of pages for this subreddit.
permission_level number

Determines who should be allowed to access and edit this page 0 indicates that this subreddit's default wiki settings should get used, 1 indicates that only approved wiki contributors on this subreddit should be able to edit this page, and 2 indicates that only mods should be able to view and edit this page.
